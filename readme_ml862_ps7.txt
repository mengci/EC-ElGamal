This is the directory for Problem set 7, CPSC 467, Fall 2013.

Mycode Directory contains 6 commands required.

Compile command: g++ -Wall -g -o mycode mycode.cpp -Iinclude -Llib ../lib/libps7.a -lgmp -lgmpxx

Execute command:./encode params-P192.ec dat.txt
		./genkey params-P192.ec key 1234
		./encryptn key.pub dat.pt dat-num.ct 1234
		./decryptn key.prv dat-num.ct dat-num-decrypted.pt
		./encrypt key.pub dat.txt dat-txt.ct 1234
		./decrypt key.prv dat-txt.ct dat-txt-decrypted.txt

Files in Mycode Directory:
		key.pub and key.prv are key files generated by command genkey;
		dat-num.ct generated by command encryptn contains the ciphertext for dat.pt;
		dat-num-decrypted.pt generated by command decryptn contains the decrypted message for dat-num.ct;
		dat-txt.ct and dat-txt-decrypted.txt are ciphertext and decrypted message for dat.txt.
